- 投票
    > !t vote title option1 option2 option3...

    最多可以到option10 

    ![vote](img/vote.png)

    當option只有一個時會變成贊成與不贊成

    ![vote_one](img/vote_one.png)

    目前每個禮拜一會詢問當周桌遊夜時間

    ![Night](img/Night.png)
    
    並且可以透過
    > !t getResult

    > !t gr
